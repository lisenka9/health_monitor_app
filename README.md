# Health Monitor App

Веб-приложение для управления медицинскими данными. Позволяет пользователям отслеживать артериальное давление, уровень глюкозы, вес и вести дневник самочувствия.

## Технологии

*   **Backend:** Python, FastAPI, SQLAlchemy, Pydantic, JWT
*   **Frontend:** React, TypeScript
*   **База данных:** PostgreSQL
*   **Инфраструктура:** Docker, Docker Compose

## Функциональные возможности

*   Регистрация и аутентификация пользователей.
*   Ввод и просмотр показателей: артериальное давление, глюкоза, вес.
*   Ведение дневника самочувствия.
*   Просмотр истории и фильтрация по дате.
*   Визуализация динамики показателей на графиках.

## Быстрый старт

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка на ваш репозиторий>
    cd health_monitor_app
    ```

2.  **Создайте файл с переменными окружения:**
Скопируйте пример файла и при необходимости отредактируйте значения.
    ```bash
    copy .env.example .env
    ```
3.  **Запустите приложение:**
    ```bash
    docker-compose up -d

4.  **Проверьте работоспособность:**
